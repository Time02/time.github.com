比特币sv节点 文件说明

2020-08-02

bitcoin bsv node

本次使用的比特币 sv 版本为 1.0.4：bitcoin-sv

banlist.dat：存储被禁节点的 IP /子网

bitcoin.conf：bitcoind 的配置文件

bitcoind.pid：运行时存储 bitcoind 的进程 ID

blocks/blk000??.dat：块数据（默认每个文件128 MiB）

blocks/rev000??.dat：撤销数据块

blocks/index/*：块索引 (LevelDB)

chainstate/*：区块链状态数据库 (LevelDB)

database/*：BDB数据库环境

db.log：钱包数据库日志文件

bitcoind.log：包含调试信息和由 bitcoind 生成的常规日志记录

mempool.dat：内存池事务转储

peers.dat：节点 IP 地址数据库

wallet.dat：具有密钥和交易的个人钱包（BDB）

.cookie：会话RPC身份验证cookie（使用cookie身份验证时在启动时编写，在关闭时删除）

onion_private_key：缓存的Tor隐藏服务私钥
